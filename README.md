# Portfolio-Simulator

Modernes React-Frontend (Vite + TypeScript) zur Simulation eines Investment-Portfolios mit Zinseszins (Beispielrenditen).

## Start
1. In `portfolio-simulator` ein Terminal öffnen.
2. `npm install`
3. `npm run dev`
4. Browser öffnen: `http://localhost:5173`

## Hinweise
- Die Renditen sind **vereinfachte Beispielwerte** (keine Finanzberatung).
- Portfolios (inkl. Gewichten, Startkapital, Laufzeit) werden optional in `localStorage` gespeichert.
